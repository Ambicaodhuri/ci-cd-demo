# 🚀 Python Application with Docker & CI Setup

## 📌 Project Overview
This project demonstrates a Python-based application with Docker containerization and CI/CD pipeline setup using GitHub Actions. It ensures smooth development, testing, and deployment processes.

---

## 🛠️ Tech Stack
- Python
- Docker
- GitHub Actions (CI/CD)
- Pytest (Testing)

---

## ⚙️ Features
- Containerized Python application using Docker
- Automated testing using Pytest
- Continuous Integration setup using GitHub Actions
- Easy deployment and scalability

---

## 📂 Project Structure
- `app.py` → Main application
- `test_app.py` → Test cases
- `Dockerfile` → Container setup
- `requirements.txt` → Dependencies
- `.github/workflows` → CI pipeline
